# bitlabs
BitLabs offers the opportunity to monetize your app with rewarded surveys easily.

## Installation

```sh
npm install bitlabs
```

OR

```sh
yarn add bitlabs
```

## Usage

Check Refer to the full guide: [React Native SDK](https://developer.bitlabs.ai/docs/react-native-sdk#list-of-survey-widgets)
